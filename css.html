Selector de hijos

h1.principal {font-family:Arial;}

Se aplica el tipo de fuente Arial a todos los elementos h1 que tienen definida la clase "principal"

h1#titulo1 {font-family:Arial;}

Se aplica el tipo de fuente Arial al elementos h1 que tiene definidida la propiedad id con el valor "titulo1".

ul em{color:#f00;}

Todos los elementos "em" se mostraran de color rojo siempre y cuando estén contenidos por un elemento "ul".

h1>em {color:#0f0;}

Aquí solo se aplica el color #0f0 si em es un elemento hijo directo de h1.

Select. hermanos

<body>
<h1>Titulo 1</h1>
<p>Esto es el primer párrafo.</p>
<p>Esto es el segundo párrafo.</p>
<p>Esto es el tercer párrafo.</p>
</body>

h1+p{color:#f00;}

Seleccionamos el primer párrafo que le sigue al elemento h1. Luego el primer párrafo se pinta de rojo.

p+p{color:#f00;}

Se pintan de rojo el segundo y tercer párrafo ya que a cada uno de estos párrafos le antecede otro párrafo (no se pinta el primer párrafo porque no le antecede un párrafo sino un "h1")

h1~p{color:#f00;}

Como resultado se aplica el color rojo a todos los párrafos adyacentes que tienen como hermano el elemento "h1".

Select. atributos

a[target]{color:#f00;}

Esta regla se aplica a todos los elementos de tipo "a" que tienen definida la propiedad llamada "target".

a[href="#"]{text-decoration:line-through;}

Esta regla se aplica a todos los elementos de tipo "a" que tienen almacenados en la propiedad href la cadena "#".

a[href^|$|*="http"] {color:#f00;}

Podemos también controlar si el valor de la propiedad comienza (^), finaliza ($) o tiene en su interior una determinada cadena de caracteres (*).

Select. child

ol li:(first/last)-child {color:green;}
(también se puede escribir como)
li:(first/last)-child {color:green;}

Selecciona el primer/último item de la lista.

tr:nth-child(odd/even) {background-color:red;}

Si a la pseudoclase nth-child le pasamos el valor odd/even nos seleccionará todos los elementos de las posiciones impares/pares.

tr:nth-child(an+b) {background-color:red;}

Selecciona la fila an+b de la tabla.
a deber ser al menos 1(no se escribiría). b es opcional.

tr:nth-last-child(an+b) {background-color:red;}

Empieza a contar desde la última fila.

Only-(child/of-type)

La pseudo-clase only-child se aplica cuando en dicho nivel hay un solo elemento.

#fechainicio li:only-child {color:blue;}

El li se pinta de rojo sólo si hay un solo elemento en ese nivel.

La pseudo-clase only-of-type se aplica cuando en dicho nivel hay un solo elemento de ese tipo (no importa cuantos otros elementos hay de otro tipo en el mismo nivel)

<div id="equipo1">
  <h1>Participantes</h1>
  <p>Pablo Martinez.</p>
</div>

#equipo1 p:only-of-type{color:red;}

Sí se pinta de rojo el párrafo ya que en ese nivel hay solo un elemento de tipo párrafo.
